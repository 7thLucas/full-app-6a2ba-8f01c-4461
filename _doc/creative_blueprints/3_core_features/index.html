What it does

Session Review

Log candidate, role, round, date, and interviewer notes in a structured form. Every session is searchable and retrievable.

Scoring Rubric

Score candidates on defined criteria — communication, technical fit, culture, role-specific competencies — so every reviewer uses the same scale.

Panel Feedback Dashboard

All reviewer scores and comments for a candidate consolidated in one view. No more chasing down opinions from three different channels.

Session History

Browse past sessions and track candidate progression across interview rounds from first screen to final decision.

Completion Tracking

See at a glance which interviewers haven't submitted feedback yet. No more wondering who's still outstanding.
